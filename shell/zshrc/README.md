## 介绍

由于默认的 zsh 配置有点麻烦。
因此用户 Robby Russel 在 GitHub 上制作了 `oh-my-zsh` 配置文件，
这是目前最流行的 zsh 配置，因为配置起来很方便，所以现在几乎成为标配。

- [ohmyz-官网]
- [ohmyz-仓库]
